# 📚 BookShop App

BookShop is a fully responsive web application where users can explore books, search dynamically, and manage their accounts with secure authentication.

It uses server-side rendering for fast performance and SEO-friendly search, while maintaining a clean and modern UI for both mobile and desktop users.

The goal of this project is to demonstrate real-world full-stack concepts like authentication flow, protected routes, server-side filtering, and responsive UI design.

---

## Features

-  Email & Google Authentication (Better Auth)
-  Dynamic book listing from API
-  Server-side search with URL params
-  Protected profile page
-  Fully responsive (mobile + desktop)
-  Clean mobile navigation menu
